# 预言机交互 (Oracle Interactions)

本节重点介绍使用预言机将链下数据带到 Starknet 区块链的概念。预言机是第三方服务，充当中介，安全地将外部数据（如资产价格、天气信息或其他现实世界数据）传输到区块链和智能合约。它还提供了实际示例和代码片段，包括如何为游戏使用
[Pragma 价格喂送](https://docs.pragma.build/Resources/Starknet/data-feeds/consuming-data)
和 [Cartridge VRF](https://github.com/cartridge-gg/vrf) 来获取可验证随机数。
